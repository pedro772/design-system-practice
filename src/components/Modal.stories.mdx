# Modals

A modal is a user experience element that comes in front of a larger parent window as a child window.

## Usage

Modals are used for specific workflows such as adding users, deleting content, sharing content, adding content, and more.

## Sign Up Modal

The Sign Up Modal is used get the user to Sign Up by clicking in a button.
